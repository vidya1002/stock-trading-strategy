# 📈 Stock Trading Strategy (NIFTY 50)

## 📌 Overview
This project builds and backtests a trading strategy using historical NIFTY 50 data.

## Features
- Moving Average (MA20, MA50)
- RSI Indicator
- Buy/Sell Strategy
- Backtesting
- Visualization

## How to Run
pip install -r requirements.txt
python main.py
